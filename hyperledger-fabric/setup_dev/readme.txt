Widnows环境配置:
1.下载 vagrant工具，并安装：https://www.vagrantup.com/downloads.html
2.下载 vitualbox并安装、运行：http://sw.bos.baidu.com/sw-search-sp/software/6bda11b7d3256/VirtualBox-5.1.30.18389-Win.exe
3.进入 fabric/devenv执行：vagrant.exe up
4.进入 虚拟机环境：vagrant.exe ssh
5.下载 fabric官方的docker image：sh download_docker_images.sh
